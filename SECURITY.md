# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in kaedevn, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please use [GitHub Security Advisories](https://github.com/mkanakureon/kaedevn/security/advisories/new) to report the vulnerability privately.

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Fix and disclosure**: Coordinated with the reporter

## Supported Versions

| Package | Version | Supported |
|---------|---------|-----------|
| `@kaedevn/interpreter` | 0.1.x | Yes |
