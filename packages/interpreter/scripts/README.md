### run_demo.ts

デモ実行用スクリプト。テストスクリプトを読み込み、モックエンジン（IEngineAPI 実装）を通じて
インタプリタで実行する。開発中の動作確認や、新しい `.ksc` スクリプトの手動テストに使用。

**実行方法:**

```bash
npx tsx scripts/run_demo.ts
```

**動作:**
1. 指定された `.ksc` ファイルを読み込む
2. IEngineAPI のモック実装を作成（コンソールに出力）
3. Interpreter インスタンスを生成してスクリプトを実行
4. セリフ、コマンド呼び出し、変数変化をコンソールに表示
